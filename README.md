# MIDITrailer
A program that visualizes a MIDI file on a piano, Synthesia style.

**Libraries Required**<br>
[Sanford.Multimedia.Midi](https://github.com/tebjan/Sanford.Multimedia.Midi)<br>
[SlimDX](https://slimdx.org/)
